选择了5篇文档，文档1234是四篇2019年全国卷高考满分作文，文档5是北京卷高考满分作文。Test文档也是2019年全国卷高考满分作文。
split.py为分词和去掉停用词程序
sim.py为文档相似度比较

TEST与文档12345用TF-IDF比较相似度 
[(0, 0.2738859), (1, 0.2668244), (2, 0.22320278), (3, 0.2797243), (4, 0.017638683)]
可以得出前四篇文档和TEST文档相似度大，最后一篇相似度差了一个数量级，和预期结果差不多。但是由于文档数较少，词典太少，导致相似度不大。

TEST与文档12345用LSI模型比较相似度
[(0, 0.99160135), (1, 0.9923636), (2, 0.9571818), (3, 0.9989278), (4, 0.021253109)]
可以得出结论和TF-IDF方法差不多，就是相似度大了很多，效果好了很多。
